# 🎂 Happy Birthday, Sonam!

## 🌐 View it live

Once GitHub Pages is enabled, the page will be live at:

**https://m2025lsp028.github.io/Sonam-s-Birthday/**

### How to enable GitHub Pages
1. Go to the repository **Settings**
2. Open the **Pages** section (left sidebar)
3. Under **Source**, select branch `main` and folder `/ (root)`
4. Click **Save** and wait ~1 minute
5. Share the link above with Sonam! 🎉
